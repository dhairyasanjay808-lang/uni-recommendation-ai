# 🎓 UniMatch AI

An AI-powered university recommendation engine that matches students to CS/AI programs based on their academic profile and interests.

## Overview
Built as a portfolio project to demonstrate practical machine learning and full‑stack development skills. The app uses **TF‑IDF vectorization** and **cosine similarity** to rank universities based on user‑provided preferences.

## Features
- Filter universities by GPA, tuition budget, and geographic region.
- Select academic interest (AI/ML, Systems, Theory, Robotics, Entrepreneurship).
- Get ranked recommendations with match scores and visual progress bars.

## Technologies
- Python, Streamlit, Pandas, Scikit‑learn (TF‑IDF, cosine similarity)

## How to Run Locally
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `streamlit run app.py`

## Dataset
The `unis.csv` file contains 13 top global CS/AI programs with attributes like GPA requirements, tuition, strengths, and acceptance rates.
